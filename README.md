# Uptime Monitor

Простой скрипт на Python для мониторинга доступности сайтов и времени ответа.

---

## 🚀 Что делает проект

Скрипт проверяет список сайтов и собирает информацию:

- доступен сайт или нет
- HTTP статус код
- время ответа (в миллисекундах)
- общий процент доступности (uptime)

Результаты выводятся в консоль и записываются в файл `monitor.log`.

---

## ⚙️ Возможности

- Проверка доступности сайтов
- Измерение времени ответа
- Обработка ошибок (таймаут, отсутствие соединения)
- Подсчёт uptime (%)
- Логирование в файл
- Бесконечный цикл мониторинга
